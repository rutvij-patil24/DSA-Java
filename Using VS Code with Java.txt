General Keyboard Shortcuts for Java in VS Code:

1. Run the program: Ctrl + F5 (Windows/Linux), Cmd + F5 (Mac)
Run with Debugger: F5

2. Show IntelliSense suggestions: Ctrl + Space (Windows/Linux), Cmd + Space (Mac)

3. Go to Definition: F12
Jumps to the definition of a variable, method, or class.

4. Peek Definition: Alt + F12
Shows the definition inline without navigating away.

5. Go to Implementation: Ctrl + F12 (Windows/Linux), Cmd + Option + B (Mac)
Jumps to the implementation of a method or interface.

6. Find References: Shift + F12
Lists all references to a method, variable, or class.

7. Quick Fix / Auto Import: Ctrl + . (Windows/Linux), Cmd + . (Mac)
Suggests fixes, including auto-imports for missing classes or packages.

8. Rename Symbol: F2
Renames variables, methods, or classes across the codebase.

9. Format Code: Shift + Alt + F (Windows/Linux), Shift + Option + F (Mac)
Formats the current file or selected code block.

10. Toggle Comment: Ctrl + / (Windows/Linux), Cmd + / (Mac)
Toggles single-line comment.

11. Block Comment/Uncomment: Shift + Alt + A (Windows/Linux), Shift + Option + A (Mac)
Toggles block comments for selected lines.

12. Find in File: Ctrl + F (Windows/Linux), Cmd + F (Mac)
Finds occurrences in the current file.

13. Replace in File: Ctrl + H (Windows/Linux), Cmd + Option + F (Mac)
Replaces text in the current file.

14. Go to File: Ctrl + P (Windows/Linux), Cmd + P (Mac)
Quickly opens files by typing their name.

15. Navigate Between Open Files: Ctrl + Tab (Windows/Linux), Cmd + Tab (Mac)
Switches between open files.
